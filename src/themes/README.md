# `themes/`

Chaque theme doit devenir autonome :

```text
src/themes/<theme>/
  theme.config.ts
  cards.data.ts
  vehicles.data.ts
  events.data.ts
  shop.data.ts
  ui.config.ts
  audio.config.ts
  README.md
```

Le theme fournit du contenu et une direction visuelle. Il ne modifie pas directement les regles du moteur.
