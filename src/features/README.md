# `features/`

Une feature = une capacite utilisateur ou gameplay identifiable.

Chaque dossier de feature doit contenir, a terme :

- composants locaux ;
- hooks locaux ;
- styles locaux ;
- tests ;
- README court ;
- aucune logique globale cachee.

Exemple : `features/hand/` possede la main, le hover, la selection et le drag/drop. Elle ne possede pas les effets des cartes.
