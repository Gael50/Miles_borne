# `entities/`

Objets metier du jeu.

Sous-domaines :

- `cards/` : definitions, normalisation, effets, descriptions.
- `players/` : etat joueur, main, inventaire, statut, carburant.
- `vehicles/` : vehicules/skins, bonus/malus, filtrage.
- `events/` : evenements aleatoires, descriptions et effets.
- `shops/` : PNJ, catalogues, achats.

Les entites ne doivent pas connaitre les ecrans qui les affichent.
